## Introducción
Este es un sistema completo de autenticación con Firebase, React, Vite and Chakra-UI. La app no cuenta con ninguna funcionalidad por si misma, sino que actua como plantilla preconfigurada para cualquier proyecto que use la autenticación de Firebase.

## Funcionalidades
Desarrollar aplicaciones con autenticación siempre lleva bastante tiempo. Sin embargo, una parte considerable de ese tiempo se podría ahorrar usando una plantilla ya configurada para iniciar los proyectos.

- Frontend básico con React. Usa Vite en lugar de webpack como empaquetador, debido a su gran optimización.
- Sistema de registro, inicio y cierre de sesión con Firebase Auth usando email y contraseña.
- Rutas protegidas específicas para usuarios loggeados, y otras para usuarios sin sesión iniciada.
- Ruta básica de perfil de usuario, con opción de cambiar el email asociado a la cuenta.
- Recuperación de contraseña perdida mediante email.
- Contexto de React ya configurado para almacenar posible información adicional en la sesión del usuario.
- Tema oscuro por defecto configurado con Chakra-UI.
