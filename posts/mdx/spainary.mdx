## Introducción
Spainary es un proyecto que inicié para aprender sobre la generación de sitios estáticos (SSG) de NextJS. Se trata de un blog que utiliza archivos mdx para la gestión del contenido, internacionalización (actualmente en 3 idiomas), un sistema propio de tracking de visitas y minijuegos formato trivial.

## Funcionalidades
- Creación de posts en markdown extendido para ser generados como páginas estáticas al momento de desplegar la web
- Trackeo de visitas y compartidos en una base de datos Mongo
- Generación de contenido en múltiples idiomas
- Trivials
